# Portfólio - Monalysa Silva

Este é o meu site de portfólio, desenvolvido como parte dos meus estudos em Ciência da Computação. O objetivo do projeto é apresentar um pouco sobre mim, minhas habilidades, meus interesses profissionais e meus projetos.

## Sobre o projeto

O site foi criado utilizando **HTML**, **CSS** e **JavaScript**, com uma estrutura simples e organizada. Ele possui páginas de navegação, seções informativas e links para contato profissional.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Bootstrap Icons

## Funcionalidades

- Página inicial com apresentação pessoal
- Página sobre mim
- Página de projetos
- Botão com link para o LinkedIn
- Botão de contato por e-mail
- Layout organizado com Flexbox
- Uso de JavaScript para adicionar interatividade ao site

## Estrutura do site

O portfólio possui as seguintes páginas:

- `index.html` - Página inicial do portfólio
- `sobre.html` - Página com informações sobre mim
- `projetos.html` - Página destinada aos meus projetos
- `style.css` - Arquivo de estilos do site
- `script.js` - Arquivo com recursos em JavaScript
